Практическое задание по теме “Введение в проектирование БД”

1. Создать структуру БД Вконтакте по скриптам, приложеным в файле примеров examples.sql

Созданная структура находится в файле vk_DB_structure.sql

2. Используя сервис http://filldb.info или другой по вашему желанию, сгенерировать тестовые данные для всех таблиц, 
учитывая логику связей. Для всех таблиц, где это имеет смысл, создать не менее 100 строк. 
Загрузить тестовые данные. Приложить к отчёту полученный дамп с данными.

Дамп с данными находится в файле vk_sample.sql

Развертываем дамп в баззу данных vk. Для этого вводим следующие команды:

andrey@ubuntu:~$ nano vk_sample.sql
Создаём файл в Ubuntu и копируем в него все данные из дампа при помощи редактора nano

andrey@ubuntu:~$ mysql vk < vk_sample.sql
развертываем дамп в базу данных vk
